# 1.3 计算机的硬件指标

1. 机器字长
    * CPU一次能够处理的数据的位数
    * CPU寄存器一般等于机器字长
2. 运算速度
   * CPU主频: 时钟频率
   * CPU核心数和支持线程数
   * CPI: 执行一条指令所需要的时钟周期
   * IPC: 每周期能执行完成的指令数
   * MIPS: 每秒可执行百万条指令
   * FLOPS: 每秒可执行的浮点运算
3. 存储容量
   * 主存容量
        * MAR MDR表示: MAR 10, MDR 8 :: 1K×8位 
        * 字节数 1KB, 1GB
   * 辅存容量: 1TB
